2. 切换到命令提示窗口, 在命令提示符下输入 `nslookup www.baidu.com`, 分析执行结果：

    `nslookup` 是一个用于查询 DNS（域名系统）的网络管理工具，它允许用户查找域名与 IP 地址之间的映射关系，或者查找 DNS 服务器的相关信息。

    执行结果如下：

    ![alt text](image-1.png)

   1. **服务器**：
      - 服务器地址是 `10.8.8.8`，这通常是本地 DNS 服务器的地址。

   2. **非权威应答**：
      - 这意味着 DNS 服务器不是权威服务器（即不是域名的原始注册服务器），但仍然提供了查询结果。
       - **权威应答**：由域名的权威 DNS 服务器提供，这些服务器包含域名的官方记录。
      - **非权威应答**：由非权威 DNS 服务器提供，这些服务器可能缓存了域名的记录，但不是最新的。
   3. **名称**：
      - `www.a.shifen.com` 可能是 `www.baidu.com` 的一个别名或 CNAME 记录。

   4. **地址**：
      - 显示了多个 IP 地址，包括 IPv6 和 IPv4 地址：
        - IPv6 地址：`2409:8c54:870:67:0:ff:b0c2:ad75` 和 `2409:8c54:870:34e:0:ff:b024:1916`
        - IPv4 地址：`183.240.98.161` 和 `183.240.98.198`

   5. **别名**：
      - `www.baidu.com` 表示查询的域名。
      - 别名（CNAME）记录是一种 DNS 记录，它将一个域名映射到另一个域名。在这种情况下，`www.a.shifen.com` 可能是 `www.baidu.com` 的一个别名。
